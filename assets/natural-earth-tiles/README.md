Natural Earth II raster tiles for Fly.

Source: Natural Earth 1:10m raster, NE2_HR_LC_SR_W.
Downloaded from https://naciscdn.org/naturalearth/10m/raster/NE2_HR_LC_SR_W.zip

The original raster is public domain Natural Earth data. These 512px JPEG tiles
were generated from the 21600 x 10800 source so the game can load nearby map
detail without decoding one massive world image in the browser.
